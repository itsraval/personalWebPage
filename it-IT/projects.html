DESCRIZIONE

Questo è un password manager locale con interfaccia grafica.

Che cosa puoi fare!

Puoi salvare tutti i tuoi account su una singola app. Usa il generatore di password per creare delle password complesse.
Tutti i dati sono salvati UNICAMENTE sul tuo dispositivo. L'applicazione utilizza hash SHA256 e il cifrario AES. Se qualcuno prova ad entrare viene bloccato e puoi ricevere un'email di notifica.
Facile e veloce da usare grazie all'interfaccia grafica. Applicazione e manuale sul funzionamento unicamente in inglese.

HTML-SRC

Download
Code </>

DESCRIZIONE

Questo programma controlla, da quando è stato avviata, se una pagina web è stata aggiornata.

Che cosa puoi fare!

"html-src" controllerà ogni 5 minuti l'url che hai richiesto, fino a che la pagina web non sarà aggiornata o fino a quando non chiuderai l'app.
Puoi salvarti i link (in app), in modo tale da non doverli riscrivere ogni volta che vuoi controllare le pagine.
Puoi cancellare i link di cui non hai più bisogno (in app).
Linguaggio: Python 3.8
